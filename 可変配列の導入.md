## 2次元配列
### 定義

vector<vector< "型" >> "配列名" ( "行数" , vector< "型" >( "列数" ) );

### サイズ
行サイズ　　

    "配列名" .size();　

列サイズ　　

    "配列名"　[0].size();

### サイズ変更
    "配列名" .resize( "行数" );
	  for (unsigned int i=0;i< "列数" ;i++){
		  "配列名" [i].resize( "列数" );
	  }
列のサイズを変更するときは、すべての行に対して列の変更を行う必要がある
